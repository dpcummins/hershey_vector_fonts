# Hershey Vector Fonts
## A C library for loading Hershey vector font definitions, and some C functions for drawing the character sets
### Description
**A C library for loading Hershey vector font definitions, and some C functions for drawing the character sets**

### Files
- hershey_font.h
- showfont.c
- *.hf

### Example usage
- Compile:
  `gcc showfont.c -lX11 -o showfont`
- Run:
  `./showfont simplexroman`
