# Hershey Vector Fonts
## A C library for loading Hershey vector font definitions, and some C functions for drawing the character sets using XLib
### Description
**A C library for loading Hershey vector font definitions, and some C functions for drawing the character sets using XLib**

### Files
- hershey_font.h
- showfont.c
- *.hf

### Example usage
- Compile:
  `gcc showhfont.c -lX11 -o showhfont`
- Run:
  `./showhfont simplexroman`
